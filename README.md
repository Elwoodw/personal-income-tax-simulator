## 下载安装
-  **安卓/Android版：** [点击下载安卓app](https://gitee.com/jackwjc/personal-income-tax-simulator/releases/download/v1.0/geshui.apk)
-  **苹果ios版：**   [点击查看苹果安装教程PDF](https://gitee.com/jackwjc/personal-income-tax-simulator/releases/download/v1.1/jiaocheng.pdf)
-  **PC版:** 先下载雷电模拟器，然后在雷电模拟器中安装**安卓app**运行 [点击下载雷电模拟器](https://lddl01.ldmnq.com/downloader/ldplayerinst9.exe?n=ldplayer9_ld_6000_ld.exe)
## 使用方式

安装后建议先观看操作教程视频，然后试用模拟器，如需购买卡密点击卡密购买，在平台购买账号密码，输入账号密码即可使用本软件。
