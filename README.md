## 使用方式
安装后点击购买，在平台购买卡密，输入卡密激活即可使用本软件。
- 激活码一次性使用
- 激活后立即失效
- 一机一码，绑定设备
- 软件更新方法：断网打开app，此时进入验证界面，过几秒联网，点击检查更新，如果有新版本则下载，卸载原有安装包，重装新版，如果不想更新直接关闭软件重新进入即可。
